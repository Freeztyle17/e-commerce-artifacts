## 1. Контекст и границы

### 1.1 Описание системы

**Название:** Система управления складом для e-commerce (WMS-lite)  

**Назначение:** Лёгкая система класса WMS для автоматизации складских процессов интернет-магазина. Обеспечивает приёмку товаров, размещение, обработку заказов (комплектация, упаковка, отгрузка), управление остатками в реальном времени и интеграцию с платформой e-commerce.
